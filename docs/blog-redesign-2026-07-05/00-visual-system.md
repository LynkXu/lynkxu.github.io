# Visual System

## Subject

这是一个以长期阅读为核心的个人博客。读者进入后应该先看到内容，而不是主题展示、标语或功能介绍。

## Direction

采用 editorial ledger：左侧固定站点栏，右侧为阅读区。整体像一份克制的私人目录，靠文字、细线、日期、节奏建立美感。

## Tokens

- Background: `#fbfaf7`，近白纸面，不做渐变和装饰图形。
- Ink: `#242321`，标题和正文主色。
- Muted: `#706d66`，日期、说明、侧栏文本。
- Rule: `#dedbd2`，所有分割线。
- Accent: `#3f6f5a`，仅用于链接 hover、当前导航和极少量数值。
- Dark background: `#151514`，暗色模式仍保持低饱和。

## Typography

- 正文继续使用现有 sans stack，避免中英混排过度文学化。
- 标题使用同一字族但更低字重，少量放大，不使用海报式大字。
- 文章正文宽度控制在 `68ch` 左右，列表页比文章页更紧凑。
- 所有 letter spacing 保持 0，只有小标签允许极小 uppercase 间距。

## Layout

Desktop:

```text
| left rail 220px | main reading area 680-880px | optional context 220px |
```

Mobile:

```text
| compact top identity |
| horizontal nav       |
| content              |
| context              |
```

## Signature

用“日期/年份作为左侧细窄锚点”的 ledger 结构贯穿：首页最新文章、归档、标签、说说都以时间锚点帮助阅读扫描。

## Must Not

- 不使用大卡片堆叠、渐变、装饰背景、彩色徽章墙。
- 不新增 slogan 或解释博客功能的文字。
- 不修改摄影、旅行、运动、媒体、工具、版权等二级功能页，只在导航保留入口。
- 不改内容数据、同步脚本、评论、点赞、LightGallery、Mastodon/NeoDB/Strava 数据来源。
