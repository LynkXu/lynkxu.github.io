# About Page Design

## Page

`/about`

## Job

保留现有个人介绍内容，但用更安静的排版呈现，而不是宣传页。

## Content To Keep

- 现有自我描述、联系方式、SponsorAbout、Now、Blog History。

## Layout

- 主区是一篇短介绍文，不使用 emoji hero 或口号式标签墙。
- 三个个人片段降级为普通定义列表。
- 联系方式和 sponsor 放在主区末尾，保持低调。
- 右侧 context 放 Now 和 Blog History，使用细线时间列表。

## Acceptance

- 页面像个人说明，不像 landing page。
- 不出现“功能说明”或重复性人格标签。
- Sponsor 不抢主内容。

## Final Review

- Removed emoji hero treatment and kept the existing personal introduction, contact, sponsor, now, and blog history content.
- Reworked fragments and contact/support areas into line-based definition/list sections.
- Verified desktop, mobile, and dark desktop screenshots. No horizontal overflow.
