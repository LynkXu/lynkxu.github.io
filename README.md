## LynkXu's Blog

> 基于 Astro 构建的个人博客/个人站点（静态生成），部署在 GitHub Pages。

- **Online**: <https://lynkxu.github.io>

<!-- ![Preview](./public/banner.webp) -->

## 开发

| Command       | Action                                 |
| :------------ | :------------------------------------- |
| `npm install` | 安装依赖项                             |
| `npm run dev` | 在 `localhost:4321` 启动本地开发服务器 |
| `npm run build` | 构建生产站点到 `./dist/`             |
| `npm run preview` | 预览本地构建                       |

## 基于模板改造

本项目基于我使用的博客模板 **`@anghunk/astro-blog`** 改造而来（模板作者站点：<https://zishu.me>）。  
仓库中部分实现与结构、以及最初的 `README` 内容来自该模板，在此致谢。

（模板作者也有参考 [Taking regular screenshots of my website](https://alexwlchan.net/2024/scheduled-screenshots/) 通过 GitHub Actions 定期截图记录站点变化；相关截图仓库见 <https://github.com/anghunk/scheduled-screenshots>。）
